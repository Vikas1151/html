💡 Concepts

HTML & CSS ke Zaroori Concepts

Sabse important cheezein ek jagah pe samjho

🏷️

HTML Tags

Tags se webpage ka dhanchaa banta hai. Jaise <h1> heading ke liye, <p> paragraph ke liye, <div> container ke liye.

🎨

CSS Selectors

CSS selectors batate hain kis element ko style karna hai — tag, .class, ya #id se select karte hain.

📦

Box Model

Har element ek box hai — content, padding (andar space), border (rekha), margin (bahar space). Yeh samajhna bahut zaroori hai!

📐

Flexbox

Flexbox se items ko side-by-side, center mein, ya space ke sath arrange karte hain. display:flex likh do aur magic dekho!

🗂️

CSS Grid

Grid se rows aur columns mein layout banate hain — newspaper ya magazine layout ke liye perfect!

📱

Responsive Design

@media queries se website mobile, tablet, laptop — sab par sahi dikhti hai. Ek code, sab jagah perfect!

✨

Animations

@keyframes aur transition se elements bounce, fade, rotate, slide kar sakte hain — bilkul games ki tarah!

🔤

Google Fonts

Google Fonts se 1000+ fonts free mein milte hain. @import se website mein add karo aur font-family se use karo!

HTML Structure
<!DOCTYPE html> <html> <head> <title>Title</title> </head> <body> Content yahan </body> </html>
CSS Selector Types
/* Tag */ p { color: blue; } /* Class */ .card { background: white; } /* ID */ #logo { font-size: 24px; }
Flexbox Center
.box { display: flex; justify-content: center; align-items: center; gap: 16px; }
CSS Grid
.grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 16px; }
@keyframes Animation
@keyframes bounce { 0%,100% { transform: translateY(0); } 50% { transform: translateY(-20px); } } .ball { animation: bounce 1s infinite; }
Media Query (Responsive)
@media (max-width: 600px) { .grid { grid-template-columns: 1fr; } }
